# Story 001: ランチルーレット基本機能

## Why（なぜ作るか）
チームのランチ先をみんなで楽しく決められるように、お店一覧とルーレットで1件選べるWebアプリを作りたい。

## What（機能要件）

### 必須要件
- [x] 5件のお店を一覧表示する
- [x] ルーレットを回してランダムに1件選べる
- [x] 選ばれたお店がハイライト表示される
- [x] リロードしてもお店データが保持される

### 受け入れ条件
1. お店一覧が画面に表示される
2. 5件のお店が登録されている
3. ルーレットでランダムに1件選べる
4. リロードしてもデータが保持される

## How（技術制約）
- Next.js 14（App Router）+ TypeScript + Tailwind CSS
- データ保存は localStorage のみ（DB禁止）
- `src/data/restaurants.ts` に初期データを定義
- localStorage アクセスは `lib/storage.ts` 経由に統一

## Edge Cases
- localStorage が無効化されている場合 → 初期データで表示
- ルーレット回転中に再度ボタンを押した場合 → 無視する
- ブラウザバック・リロード時にデータが消えないこと
