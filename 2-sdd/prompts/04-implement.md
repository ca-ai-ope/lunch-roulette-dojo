# Step 4: Implement（15分）— 実装

- SPEC.md は記入済みですか？
- CLAUDE.md は作成済みですか？（Step 1 参照）

## Claude Codeへのプロンプト

スキルを使う場合：
```
/implement
```

手動でプロンプトを入力する場合：
```
SPEC.md に従って投票機能をリファクタリング（正しく作り直し）してください。
既存のルーレット機能は壊さないこと。
実装後、SPEC.md の What セクションのチェックボックスを更新してください。
```

## 注意
- CLAUDE.md を作成していれば、AIは自動で SPEC.md を参照します
- CLAUDE.md がない場合、プロンプトで毎回 SPEC.md を指定する必要があります
