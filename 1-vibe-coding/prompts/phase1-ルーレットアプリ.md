# Phase 1: Vibe Coding — ランチルーレットアプリ

## メインプロンプト（これをClaude Codeにそのままコピペ）

```
ランチのお店を5件登録できて、ルーレットで1件選ぶ
Webアプリを作って。見た目もいい感じで。
```

## 別パターン（好きなのを選んでOK）

```
ランチ決めるの面倒だから、お店リストからランダムに1つ選んでくれるWebアプリ作って。
```

```
チームのランチ先を決めるルーレットアプリ作って。お店を何個か登録できて、
ボタン押したらランダムで1つ選ばれるやつ。見た目もいい感じで。
```

## ポイント
- **仕様は書かない。** 思いついたまま伝える
- **「いい感じ」「オシャレに」** など曖昧な指示でOK
- 技術スタックやデータの保存方法は指定しない — AIに任せる
- 動いたら成功！コードの中身は気にしない

## ⏱ 制限時間: 10分

## 振り返り（10分・チームで議論）
- **速かった？** 最初に動いたのは何分後？
- **コード説明できる？** 生成されたコードの中身を理解している？
- **明日続きを書ける？** 引き継ぎドキュメントはある？
- **このまま機能追加できそう？** → Phase 2で試してみよう
